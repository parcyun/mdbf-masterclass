아임웹 페이지에 한 번만 — iframe 자동 리사이즈

Listener 1회 설치
전역 리스너 (아임웹 페이지 헤더에 한 번만)
이 스크립트를 아임웹 페이지 어딘가에 한 번만 붙여넣으면, 모든 임베드 iframe의 height가 자동 조정됩니다. 페이지마다 height를 지정할 필요 없음.
전역 리스너 — 아임웹 페이지에 한 번만 추가
<script> addEventListener('message', function(e){ if (!e.data || e.data.type !== 'mcResize') return; var fs = document.getElementsByTagName('iframe'); for (var i = 0; i < fs.length; i++) { if (fs[i].contentWindow === e.source) { fs[i].style.height = e.data.height + 'px'; break; } } }); </script>
